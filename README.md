# 🚛 Special-Trans — Waste Removal & Aggregates Delivery

🔗 Live Demo: [https://your-link-here](https://dimainit.github.io/spec-tranc-site/)

High-performance landing page for a transportation company, featuring dynamic routing, multilingual support, and scalable UI architecture.

## 🎯 Overview

This project focuses on building a scalable, SEO-friendly interface with complex UI sections such as equipment rental and material catalogs, optimized for a mobile-first experience.

## 🛠 Tech Stack

- JavaScript  
- SCSS  
- Vite  
- Page.js  
- i18next  

## 🚀 Features

- Fully responsive layout (360px → 1920px)  
- Client-side routing (SPA navigation without reloads)  
- Multilingual interface (EN / UA / RU)  
- Interactive UI components (modals, burger menu, forms)  
- Optimized assets and performance via Vite  

## 💡 Highlights

- SPA architecture with custom routing logic  
- Clean and modular SCSS structure  
- Multi-language implementation with i18next  
- Focus on performance and UI scalability
